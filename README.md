# Face-Recognize
本项目是一个基于Django的网页应用，整合了MTCNN进行精准的人脸检测与对齐，FaceNet提取面部特征，以及SVM实现高效分类，提供了一套完整的人脸识别解决方案。

## Environment
Python 3.6 

TensorFlow 1.7

Django: (ensure the latest version compatible with Python 3.6 and TensorFlow 1.7)
## DataSet
ORL Database of Faces
